Privacy Policy

Last updated: March 7, 2026

📋 Overview

MailHarvest Pro is a browser extension that extracts email addresses from webpages and PDF documents. We are committed to protecting your privacy.

✅ MailHarvest Pro does not collect, transmit, or store any personal data on external servers. All data remains exclusively on your device.

💾 Data Storage

All extracted email addresses are stored locally on your device using your browser's built-in local storage (chrome.storage.local). This data never leaves your browser and is never sent to any server, database, or third party.

You have full control over this data at all times. You can clear it instantly using the "Clear" button inside the extension.

🌐 Network Activity

MailHarvest Pro does not make any network requests to external servers. The extension operates entirely within your browser. The only network activity is the normal page loading you initiate yourself when browsing websites.

📄 PDF Processing

When extracting emails from PDF documents, all processing happens locally in your browser using an embedded PDF library (PDF.js). PDF contents are never uploaded or transmitted anywhere.

🔐 Permissions

MailHarvest Pro requests the following browser permissions:

storage — To save extracted emails locally on your device
activeTab — To read the content of the page you are currently viewing
scripting — To run the email extraction on the active page
tabs — To manage automation across multiple URLs you provide

These permissions are used solely to provide the extension's core functionality and for no other purpose.

👤 Personal Information

We do not collect, store, sell, or share any personal information about our users. We do not use analytics, tracking pixels, or any form of user monitoring.

🔄 Changes to This Policy

If we make material changes to this privacy policy, we will update the "Last updated" date above. Continued use of the extension after changes constitutes acceptance of the updated policy.
